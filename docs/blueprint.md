# **App Name**: VisionForge

## Core Features:

- Image Input & Preview: Allow users to securely upload images for analysis and display them within a dynamic preview panel with zoom and pan capabilities.
- YOLOv8 Object Detection Integration: Process uploaded images using YOLOv8 for automated object identification, visualizing detections with toggleable bounding boxes directly on the image preview.
- Intelligent Image Analysis Tool: Utilize a generative AI tool (Gemini LLM) to analyze images, incorporating YOLOv8 detection data, to produce a comprehensive scene description, emotional analysis, photography issues, and potential symbolism in a structured JSON format.
- AI-Driven DIP Suggestion Engine: Leverage the Gemini LLM as a tool to generate tailored digital image processing (DIP) suggestions, complete with specific operations, justifications, and detailed parameters to guide users toward optimal image enhancements.
- Interactive DIP Application with Real-time Feedback: Present AI-generated DIP suggestions allowing users to individually apply operations and instantly preview changes to the image with real-time updates.
- Comprehensive Manual Image Editing Tools: Provide a rich set of user interface controls (sliders, buttons) for direct manual application of various filters and adjustments, such as brightness, contrast, gamma, and sharpness.
- Enhanced Image Export: Enable users to download the final, processed image in a high-quality format after all applied digital image processing operations.

## Style Guidelines:

- Color scheme: Dark theme, reflecting professionalism and ideal for image viewing. The palette focuses on deep, cool tones to convey intelligence and precision, while ensuring optimal contrast for readability.
- Primary color: A rich, muted blue-purple (#6963DB) for interactive elements, primary text, and emphasis. This tone is chosen to suggest innovation and sophistication.
- Background color: A very dark purple-grey (#18171C), visibly derived from the primary hue but heavily desaturated, providing a quiet canvas for image display and UI elements.
- Accent color: A vivid light blue (#78D2F8) used for highlights, calls-to-action, and critical feedback. This analogous hue contrasts effectively with the primary and background, adding visual interest without being distracting.
- Headline font: 'Space Grotesk' (sans-serif) to impart a modern, techy, and scientific feel suitable for titles and feature labels.
- Body font: 'Inter' (sans-serif) for body text, descriptions, and UI elements, chosen for its clear, objective, and neutral readability, complementing the tech-forward headline font.
- Utilize sleek, minimalist line-art icons that visually communicate complex concepts related to AI, computer vision, and image processing tools, maintaining a modern and sophisticated aesthetic.
- Implement a split-panel interface with a dominant left panel for real-time image preview and object detection toggles, and a structured right panel housing tabbed sections for AI analysis, photography issues, and enhancement tools.
- Incorporate subtle, smooth transition animations for UI elements, state changes, and especially for real-time visual feedback when applying digital image processing operations, enhancing perceived responsiveness.
